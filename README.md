## 🧠 Abstract

A conceptual framework for analysing epistemic limits in administrative export artefacts.

This repository documents the conceptual framework of Record-Surface Visibility (RSV), which examines how relational structures within administrative information systems become compressed when rendered into documentary export artefacts such as disclosure PDF bundles.

The study analyses more than 250 pages of statutory disclosure materials and identifies recurring infrastructural rendering patterns at the export boundary of administrative systems.
